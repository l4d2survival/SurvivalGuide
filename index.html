すたみな太郎グループ



 コマンド:
　・!afk, !js:
　　　観戦と参加。(away from keyboard, join survivors)

　・!bot:
　　　ボットを呼び寄せます。(It teleports all bots to you.)

　・tap F key x2(!drop):
　　　手にしているアイテムを投げます。(throw the item you are holding.)

　・tap F key x2 (!light)(観戦/死亡時 for dead players/spectators.):
　　　自分からしか見えないライトで吹き飛んだアイテムを探してあげなくっちゃ。(personal lights only you can see.)

　・!votemenu:
　　　デフォルト以外のマップで遊ぶ/アイテムの集め場所を変える(to play unofficial survival maps/set place for item)

====================================================================
　ラウンドの情報コマンド (damage stats)
　<detail> を末尾に追加するとより詳細な情報を表示します (<detail>: optional detail flag for more information)
　例(Ex.): !stats detail
--------------------------------------------------------------
　・!stats: 本人の統計 (personal summarized stat)
　・!stats all: 4人分の統計 (summarized stats for all player)
　・!stats mvp: MVP情報を含む統計 (round stats with MVP info)
　・!stats <name>: <特定のプレイヤー> 個人の統計 (stats for <specific player>)
　・!resetstats: 管理者コマンド、統計のリセット (Admin Command for resetting stats)
====================================================================

サーバーの設定

  • サバイバルの定番ポジションにアイテムを集めてあります。他のポジションやデフォルト設定にも投票できます。
  • 近接武器が遠いマップに限り近くへ移動させています。
  • ランダムアイテムのマップではアイテムを固定しています。屋上、電車、サクリ港、発電室。
  • 移動できてしまうspawn武器を固定しています。川岸。config by Killjoy
  • 床や棚に沈んで見えにくいアイテムの高さを調整して見えるようにしています。パシ港、ゲタビ。

    • 　スタック対策
  • スタックキル、いたしておりません。テスト中: ダカニコンサ、発電室。
  • 一部のマップでバリアフリーのリフォームをしています。バス停、納屋、砂糖工場、発電室。config by antena, Killjoy
